Sending XMR requires scanning your wallet's outputs on the blockchain to find spendable coins, then constructing a ring signature transaction client-side.

This needs either:

Option A: Light wallet server A server scans the blockchain using your view key and returns your outputs. Your spend key stays local.
Option B: Client-side WASM Full blockchain scanning in the browser using WebAssembly. Slow (~5-30 min first sync) but fully trustless.

We're building Option A first. Stay tuned.
